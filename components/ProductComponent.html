Description

The ProductComponent allows the buying of products, either events or tours.

It consists of four inner components:

  • ProductBreadcrumbsComponent
  • ProductDateComponent
  • ProductTimeComponent
  • ProductPricesComponent

If these are not defined, then default templates will be used.

The ProductComponent also can be customised by providing a valid JSON object to an options attribute.

<go-product product-id="" product-type="" options='{"breadcrumbs": {"enable": false}}'><go-product>

The available options are listed under the inputs section below.

The inputs for the ProductComponent can be set dynamically using a JS CustomEvent. The detail object can contain go_product_id, go_product_type and go_product_options and it must be dispatched on the go-product element. For example:

var myGoProductElement = document.getElementById('goProduct')
var event = new CustomEvent('goComponentChanges', {
  detail: {
    go_product_id: 2,
    go_product_type: 'tour',
    go_product_options: {}
}});
myGoProductElement.dispatchEvent(event);

NOTE: Do not modify the html of any components after the components script has been loaded.

Metadata

selector go-product,[go-product]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Type: ProductOptions
product-id

Available values: valid go~mus product id
product-type

Available values: event or tour

Example

<go-product product-id="1" product-type="event"></go-product>
<go-product product-id="2" product-type="tour">
  <go-product-breadcrumbs></go-product-breadcrumbs>
  <go-product-date></go-product-date>
  <go-product-time></go-product-time>
  <go-product-prices></go-product-prices>
</go-product>
<go-product-breadcrumbs *ngIf="defaults.breadcrumbs"></go-product-breadcrumbs>
<go-product-date *ngIf="defaults.date"></go-product-date>
<go-product-time *ngIf="defaults.time"></go-product-time>
<go-product-filters *ngIf="defaults.filters"></go-product-filters>
<go-product-prices *ngIf="defaults.prices"></go-product-prices>
<div #content>
  <ng-content></ng-content>
</div>
