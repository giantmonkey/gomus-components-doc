The UserComponent allows the logging in of and registering of users. It also provides the creation of guest users.

It consists of three inner components:

  • UserLoginComponent
  • UserRegistrationComponent
  • UserGuestComponent

Each inner component can also be used individually.

The UserComponent itself shows the main user screen. The User can then choose to login, register, or create guest account (needed for checkout process)

IMPORTANT: please notify us of your chosen url to redirect the user to after a successful registration so we can configure go~mus with the correct redirects.

The sub components can be selected using the url. To use append the following to the url on the page of the UserComponent:

  • #/?user=login
  • #/?user=register
  • #/?user=guest

The available options are listed under the inputs section below.

The user component itself template can be customised using the UserContentDirective (go-user-content) directive. See the second example below.

The login, registration and guest components can also be customised in this component, by defining them within the component. See the third example below.

Demos

https://components.gomus.de/user/user.html

Metadata

selector go-user
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Type: UserOptions

Examples

Example 1
<go-user></go-user>
Example 2
<go-user>
  <go-user-content>
    <div>
      <h3>Ich habe bereits ein Benutzerkonto.</h3>
      <p>Melden Sie sich einfach an.</p>
      <go-user-login></go-user-login>
    </div>
    <div>
      <h3>Ich habe kein Benutzerkonto.</h3>
      <p>Das Anlegen eines Benutzerkontos dauert weniger als eine Minute.</p>
      <button go-user-action="register">Registrieren</button>
    </div>
    <div>
      <h3>Als Gast fortfahren</h3>
      <p>Schließen Sie Ihre Bestellung als Gast ab. Es sind persönliche Angaben erforderlich.</p>
      <button go-user-action="guest">Als Gast fortfahren</button>
    </div>
  </go-user-content>
</go-user>
Example 3
<go-user options="{}">
<go-user-login></go-user-login>
<go-user-registration></go-user-registration>
<go-user-guest></go-user-guest>
</go-user>
<ng-container *ngIf="!content && !selected">
  <div>
    <h3>Ich habe bereits ein Benutzerkonto.</h3>
    <p>Melden Sie sich einfach an.</p>
    <go-user-login *ngIf="!custom.login"></go-user-login>
  </div>
  <div>
    <h3>Ich habe kein Benutzerkonto.</h3>
    <p>Das Anlegen eines Benutzerkontos dauert weniger als eine Minute.</p>
    <button go-user-action="register">Registrieren</button>
  </div>
  <div>
    <h3>Als Gast fortfahren</h3>
    <p>Schließen Sie Ihre Bestellung als Gast ab. Es sind persönliche Angaben erforderlich.</p>
    <button go-user-action="guest">Als Gast fortfahren</button>
  </div>
</ng-container>
<go-user-login *ngIf="!custom.login && (selected == 'login' || selected == 'password-reset')"></go-user-login>
<go-user-registration *ngIf="default.register"></go-user-registration>
<go-user-guest *ngIf="default.guest"></go-user-guest>
<ng-content select="go-user-login" *ngIf="custom.login && selected"></ng-content>
<ng-content select="go-user-content" *ngIf="custom.content"></ng-content>
<ng-content select="go-user-registration" *ngIf="custom.register"></ng-content>
<ng-content select="go-user-guest" *ngIf="custom.guest"></ng-content>
