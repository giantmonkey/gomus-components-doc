The UserAddresses component allows the user to add, edit and delete his/her addresses.

This component consists of two parts, the form for creating and editing addresses, and the listing of existing addresses.

  • The form can be edited by adding content directly into the component.
  • The list item can be edited by using the ng-template syntax.
  • Content can also be added to the beginning and end of the list by using the go-user-addresses-header and go-user-addresses-footer classes.

There are also a number of helper directives to provide functionality. These are listed here:

  • UserAddressesSelect
  • UserAddressesEdit
  • UserAddressesCreate
  • UserAddressesDelete
  • UserAddressesCancel

The available fields for the form can be found here: UserAddressForm.

The available properties for the address item can be found here: UserAddress

     <label>Stadt</label>

   <div go-input-container>
     <label>Land</label>

   </div>
   <div go-input-container>
     <label>Adresstyp</label>
     <select go-form-control="adress_type_id"></select>
     <div go-input-error="adress_type_id"></div>
   </div>
   <button type="submit">Speichern</button>
   <button go-user-addresses-cancel type="button">Abbrechen</button>
   <button go-user-addresses-delete type="button">Löschen</button>
   <ng-template let-item="item">
   <h2>{{item.type}}</h2>
   <ul>
   <li>{{item.adressat}}</li>
   <li>{{item.street}}</li>
   <li>{{item.zip}} {{item.city}}</li>
   <li>{{item.country}}</li>
   <li>{{item.addr_type}}</li>
   </ul>

   </ng-template>
   <h2 class="go-user-addresses-header">Adressen</h2>
   <button go-user-addresses-create class="go-user-addresses-footer" type="button">Hinzufügen</button>
