The TicketsBreadcrumbsComponent is a child of the TicketsComponent and displays the selected date and time once selected. It also allows the changing of the selected date and time by clicking on the breadcrumb.

If an ng-template is not defined then the default template is used.

Options can be specifed in the options JSON object in the TicketsComponent and the available options can be found here TicketBreadcrumbs

Metadata

selector go-tickets-breadcrumbs,[go-tickets-breadcrumbs]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Examples

Example 1
<go-tickets-breadcrumbs></go-tickets-breadcrumbs>
Example 2
<go-tickets-breadcrumbs>
  <ng-template let-date="date" let-time="time">
    <a [go-tickets-breadcrumbs-item]="date">{{date?.text}}</a>
    <a [go-tickets-breadcrumbs-item]="time">{{time?.text}}</a>
  </ng-template>
</go-tickets-breadcrumbs>
