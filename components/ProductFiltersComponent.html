The ProductPricesComponent is a child of the ProductComponent and displays the available product filters for a selected tour from the ProductComponent.

NOTE This component is only needed for tours, not events.

Options can be specifed in the options JSON object in the ProductComponent and the available options can be found here ProductFiltersOptions

Templates can be customised by providing content in the component as shown below.

  • Inputs and select fields must have the go-form-control attribute, with the go~mus model parameter name assigned to it.
  • The form element and input elements will automatically be labeled with the following status labels:
    • go-untouched: the field hasn't been blurred
    • go-touched: the field has been blurred
    • go-pristine: the input value hasn't been changed
    • go-dirty: the input value has been changed
    • go-valid: the input is valid
    • go-invalid: the input isn't valid
    • go-pending: there is asynchronous validation outstanding
  • Input container elements or form-groups in bootstrap can also receive these status labels by using the go-input-container directive.
  • Submit buttons must have the type submit. The go-submit directive will be used in the future to add more functionality.

The following control values are available:

  • product_language
  • product_age_group

Metadata

selector go-product-filters,[go-product-filters]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Examples

Example 1
<go-product-filters></go-product-filters>
Example 2
<go-product-filters>
  <div go-input-container>
    <label>Sprache</label>
    <select go-form-control="product_language" required></select>
  </div>
  <div go-input-container>
    <label>Altersgruppe</label>
    <select go-form-control="product_age_group" required></select>
  </div>
  <button type="submit">Weiter</button>
</go-product-filters>

  <form [formGroup]="form" (ngSubmit)="onSubmit(form.value)" novalidate [ngStyle]="{'display': show ? 'inherit' : 'none'}">
    <ng-container *ngIf="custom">
      <div go-input-container [class]="classes.groups">
        <label [class]="classes.labels">Sprache</label>
        <select go-form-control="product_language" required></select>
      </div>
      <div go-input-container [class]="classes.groups">
        <label [class]="classes.labels">Altersgruppe</label>
        <select go-form-control="product_age_group" required></select>
      </div>
      <button type="submit">Weiter</button>
    </ng-container>
    <ng-content></ng-content>
  </form>
