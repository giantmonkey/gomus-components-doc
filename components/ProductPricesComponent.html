Description

The ProductPricesComponent is a child of the ProductComponent and displays the available product prices for a selected date from the ProductDateComponent and a selected time ProductTimeComponent.

If an ng-template is not defined then the default template is used.

Options can be specifed in the options JSON object in the ProductComponent and the available options can be found here OptionsType

NOTE: The ProductPricesComponent can be wrapped in the ProductPricesContainerDirective to allow it to be positioned with in other elements.

<div go-product-prices-container>
  <div>
    <go-product-prices></go-product-prices>
  </div>
</div>

Metadata

selector go-product-prices,[go-product-prices]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Example

<go-product-prices></go-product-prices>
<go-product-prices>
    <ng-template let-item="item">
      <div [go-product-prices-item]="item">
        <go-select [item]="item"></go-select>
        <div>{{item?.title}}</div>
        <div>{{item?.price}} €</div>
        <div>{{item?.total}} €</div>
      </div>
    </ng-template>
    <div go-product-prices-footer>
      <go-total></go-total> €
      <button go-product-prices-action>In den Einkaufswagen</button>
    </div>
</go-product-prices>

prices: Price[]
