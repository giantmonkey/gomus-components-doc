Description

The UserToolbarComponent displays the login, registration or logout buttons.

The available options are listed under the inputs section below.

Template customisation is not yet possible, but will be coming soon.

Demo:

https://components.gomus.de/index.html

Metadata

selector go-user-toolbar
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Type: UserOptions

Example

<go-user-toolbar></go-user-toolbar>
