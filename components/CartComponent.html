Description

The CartComponent allows the displaying of a shopping cart including all of the event, tour and ticket items that have been added. It also allows the starting of the checkout process.

The CartComponent also can be customised by providing a valid JSON object to an options attribute.

Demos

https://components.gomus.de/cart/cart.html

https://components.gomus.de/cart/cart-custom.html

Metadata

selector go-cart
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Type: CartOptions

Examples

Example 1
<go-cart></go-cart>
Example 2
<go-cart>
<ng-template let-item="item">
   <div [go-cart-item]="item">
     <go-select [item]="item"></go-select>
     <button [go-cart-item-delete]="item">X</button>
     <div>{{item?.cart.title}} <br> {{item?.cart.time}}</div>
     <div>{{item?.cart.price}} €</div>
     <div>{{item?.cart.total}} €</div>
   </div>
</ng-template>
<div go-cart-footer>
   <go-total></go-total> €
   <button go-cart-action>Kostenpflichtig bestellen</button>
</div>
</go-cart>
<ng-content></ng-content>
<ng-container *ngIf="!default">
  <ng-template *ngFor="let item of cart?.items;trackBy:index;" [ngTemplateOutlet]="template" [ngOutletContext]="{item: item}">
  </ng-template>
</ng-container>
<ng-container *ngIf="default">
  <table>
    <thead>
      <tr>
        <th>Anzahl</th>
        <th>Beschreibung</th>
        <th>Einzelpreis</th>
        <th>Summe</th>
      </tr>
    </thead>
    <tbody>
      <tr *ngFor="let item of cart?.items;trackBy:index;" go-cart-item>
        <td>
          <go-select [item]="item"></go-select>
        </td>
        <td>{{item?.cart.title}} <br> {{item?.cart.time}}</td>
        <td>{{item?.cart.price}} €</td>
        <td>{{item?.cart.total}} €</td>
      </tr>
      <tr>
        <td colspan="2"></td>
        <th class="text-right">Summe</th>
        <th><go-total></go-total> €</th>
      </tr>
    </tbody>
  </table>
  <button go-cart-action>Kostenpflichtig bestellen</button>
</ng-container>
<ng-content select="[go-cart-footer]"></ng-content>
