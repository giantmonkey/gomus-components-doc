The TicketsTypeComponent is a child of the TicketsComponent and displays the available ticket types for a selected date from the TicketsDateComponent and a selected time TicketsTimeComponent.

If an ng-template is not defined then the default template is used.

Options can be specifed in the options JSON object in the TicketsComponent and the available options can be found here OptionsType

NOTE: The TicketsTypeComponent can be wrapped in the TicketsTypeContainerDirective to allow it to be positioned with in other elements.

<div go-tickets-type-container>
  <div>
    <go-tickets-type></go-tickets-type>
  </div>
</div>

Metadata

selector go-tickets-type,[go-tickets-type]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Examples

Example 1
<go-tickets-type></go-tickets-type>
Example 2
<go-tickets-type>
   <ng-template let-item="item">
     <div [go-tickets-type-item]="item">
       <go-select [item]="item"></go-select>
       <div>{{item?.title}}</div>
       <div>{{item?.price}} &euro;</div>
       <div>{{item?.total}} &euro;</div>
     </div>
   </ng-template>
   <div go-tickets-type-footer>
     <go-total></go-total> &euro;
     <button go-tickets-type-action></button>
   </div>
</go-tickets-type>
