The UserProfileComponent allows the user to view and edit his/her data, including user data, password, addresses and orders.

It consists of four inner components:

  • UserOverviewComponent
  • UserDataComponent
  • UserAddressesComponent
  • UserOrdersComponent

The UserProfileComponent itself provides a menu consisting of buttons using the UserProfileSelect directive to allow the quick and easy switching between the different profile areas.

The sub components can be selected using the url. To use append the following to the url on the page of the UserProfileComponent:

  • #/?user-profile=overview default
  • #/?user-profile=data
  • #/?user-profile=addresses
  • #/?user-profile=orders

The available options are listed under the inputs section below.

The template of the user overview component itself can be customised using the UserProfileSelect directive. See the second example below.

The template of each inner component can also be customised in this component, by defining them within the component. See the third example below.

Metadata

selector go-user-profile
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Available options: UserProfileOptions

Examples

Example 1
<go-user-profile></go-user-profile>
Example 2
<go-user-profile>
  <div>
    <button go-user-profile-select="overview">Übersicht</button>
    <button go-user-profile-select="data">Meine Daten</button>
    <button go-user-profile-select="addresses">Meine Adressen</button>
    <button go-user-profile-select="orders">Meine Bestellungen</button>
  </div>
</go-user-profile>
