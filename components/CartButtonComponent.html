The CartButtonComponent allows the displaying of the number of items in the cart and makes use of the {@link CartButtonDirective} which can be used to provide a link to the {@link CartComponent}.

The CartButtonComponent can be customised by providing a valid JSON object to an options attriabute.

Demos

https://components.gomus.de/index.html

Metadata

selector go-cart-button-container
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`

Inputs

options

Type: CartButtonOptions

Examples

Example 1
<go-cart-button-container></go-cart-button-container>
Example 2
<go-cart-button-container>
      <template let-cart="cart">
        <button [go-cart-button]="cart">{{cart.items}}</button>
      </template>
</go-cart-button-container>
<ng-container *ngIf="!default">
  <ng-template [ngTemplateOutlet]="template" [ngOutletContext]="{cart: cart}">
  </ng-template>
</ng-container>
