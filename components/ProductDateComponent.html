The ProductDateComponent is a child of the ProductComponent and displays a calendar in order to select a date for the ticket.

At the moment there is no template customisation possible, but we are working on this for a future release.

Options can be specifed in the options JSON object in the ProductComponent and the available options can be found here OptionsDate

NOTE: The ProductDateComponent can be wrapped in the ProductDateContainerDirective to allow it to be positioned with in other elements.

<div go-product-date-container>
  <div>
    <go-product-date></go-product-date>
  </div>
</div>

Metadata

selector go-product-date,[go-product-date]
*Specifies a CSS selector that identifies this directive within a template. Supported selectors include `element`, `[attribute]`, `.class`, and `:not()`
