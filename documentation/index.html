go~mus components

Description

go~mus components is a project that allows you to embed small widget-like components into your existing website to enable the buying of tickets, events and tours.

Project status

go~mus components is currently in beta and under active development. During beta, new features will be added regularly and APIs will evolve based on user feedback.

Stable and experimental components and their APIs are marked using the @stable and @experimental flags.

Getting started

Getting started is easy and simple. Simply load go~mus component javascript file with your custom config in the _gomusOptions variable.

NOTE: The config variable needs to load first before the script.

For best results, load at the end of body.

<script type="text/javascript">
  var _gomusOptions = {
    domain: "demo.gomus.de",
    locale: "de",
    payment_mode_ids: {
      default: 2
    }
  },
</script>
<script scr="http://components.gomus.de/components.js"></script>

You can then start adding components into you webpage.

Components are often made up of multiple inner components. These inner components are optional (default templates with be used instead if not specified), and any element can be used apart from for images.

Templates can be overwritten using the ng-template syntax, shown in the examples.

NOTE: To hide the contents of ng-tepmlate before go~mus components is loaded you can use css:

ng-template {
  display: none;
}

Options can be passed in the main component elements as valid json objects.

Custom tables for dynamic component parts are not possible due to strict browser rules. Flex or other css will need to be used instead.

Top level components

Main components Description Status
TicketsComponent Allows the buying of tickets
ProductsComponent Allows the buying of products (events and tours)
ProductDetailComponent Displays the details of a single product (event or tour)
ProductListComponent Displays a list of products (events and tours)
CartComponent Displays a shopping cart
CartButtonComponent Displays a link to the shopping cart with summary information
UserComponent Allows the logging in of and registering of users
UserToolbarComponent Displays login and logout buttons
CheckoutComponent Handles the checkout process and displaying of checkout states

Browser support

go~mus supports the most recent two versions of all major browsers: Chrome (including Android), Firefox, Safari (including iOS), and IE11 / Edge
