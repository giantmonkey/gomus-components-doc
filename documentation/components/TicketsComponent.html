File

src/app/tickets/tickets.component.ts

Description

The TicketsComponent allows the buying of tickets.

It consists of four inner components:

  • TicketsBreadcrumbsComponent
  • TicketsDateComponent
  • TicketsTimeComponent
  • TicketsTypeComponent

If these are not defined, then default templates will be used.

The TicketsComponent also can be customised by providing a valid JSON object to an options attribute.

<go-tickets options='{"breadcrumbs": {"enable": false}}'><go-tickets>

The available options are listed under the inputs section below.

Implements

OnInit AfterContentInit

Example

<go-tickets></go-tickets>
<go-tickets>
<go-tickets-breadcrumbs></go-tickets-breadcrumbs>
<go-tickets-date></go-tickets-date>
<go-tickets-time></go-tickets-time>
<go-tickets-type></go-tickets-type>
</go-tickets>
